## Understanding the mamarek brand

Mamarek is the personal brand of the developer Marek known unders alias mamarek123. Under this brand he plans to release projects such as video games, store page, and build his own personal brand, to serve both as a portfolio and a way to hobbystically express himself. Currently it isn't planned to include any other developers and members in this project and we arent oriented on creating revenue.
